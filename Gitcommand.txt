Git commands:
1.To view list of branches in a repo:
    Git branch
2.To create new branch 
    Git branch <branch name>
3.To checkout branch
    Git checkout <branch name>
4.To create new branch & checkout same branch
    Git checkout -b <branch name>
5.Add a specified file to repo:
    Git add <"filename">
6.To View status of files 
    Git Status
7.To commit with message
    Git Commit -m <"message">
8.To push the current branch and set the remote as upstream
    Git push --set-upstream origin <branch name>
